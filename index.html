About Me

I'm currently in the process of making a career transition into web development. Until the end of 2019 I was a self employed marine spray-painter, but I decided that that was not a fulfilling or mentally engaging enough career so after some investigation and advice I settled on web development as the path forward for me. To make that happen in the first half of 2020 I studied with Enspiral Dev Academy, a full immersion work simulation full stack web development bootcamp. I went from having no knowledge or experience with code to writing full stack JavaScript applications over the course of 15 weeks, while also learning and applying agile methodologies and team coding practices. In the course of this experience I discovered in myself a joy in solving problems, making things work and writing beautiful, functional code.

Now I am trying to take the final step into my new career by finding a position in the industry from which I can can continue to grow as a developer. In the meantime I am expanding my skillset and working on various projects.

When I'm not coding I like to read about science and history, paint models and play guitar.

Projects

NetLazy

A tool to manage a professional network, based on the spreadsheet used for that purpose by a contact of mine. The intention is to list all contacts with relevant information such as prefferred contact method, assosciated oppurutnities and frequency of contact, and to rearrange the list automatically to remind the user of needed action. Built using a full MERN stack. Currently just started and mostly unimplemented.

GitHub: github.com/tgclayton/netlazy

Game of Life

An implementation of Conway's Game of Life, using JavaScript and React. The Game itself is displayed on an html canvas element, I spent a lot of time iterating through different approaches for the display and game code in order to make the game run fast. The save/load feature is currently not working in the deployed version but works locally.

Deployed Link: tom-gol.herokuapp.com/


GitHub: github.com/tgclayton/Tom-GOL

Trial by Seven

My final project at bootcamp. A squad based combat game built over the course of five days using Phaser with a React UI wrapper. This was a fun project, I started it having no experience with Phaser at all and had to learn the basis before I could even begin. I had a lot of interesting issues come up getting the Phaser game and React UI to share information and update each other and ended up using several creative workarounds to make it happen.

Deployed Link: the-name-of-the-game.herokuapp.com/


GitHub: github.com/tgclayton/Trial-by-Seven
